### 3. **Areas**

* `GET /areas` → List all available areas.
    Response:
        [
            { "id": 1, "name": "Ikeja", "state": "Lagos" },
            { "id": 2, "name": "Surulere", "state": "Lagos" },
            { "id": 3, "name": "Yaba", "state": "Lagos" }
        ]

* `POST /areas` → Add new area (admin use).
